:::note[About]
TazUO is a modern Ultima Online client that provides enhanced features and improved performance. This guide will walk you through the installation process for different operating systems.
:::


## System Requirements

<Card title="Minimum Requirements" icon="laptop">
  <Badge text="Recommended" variant="success" /> 
  - 2GB of ram
  - Somewhat modern hardware(~2010+ should run great!)
  - .NET 9 installed [Download here](https://dotnet.microsoft.com/download/dotnet/9.0).
</Card>
